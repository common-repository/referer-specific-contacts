Referer Specific Contact plugin helps to show separate contact details for different referers.

== Description ==
Referer Specific Contact plugin allows to show different contact details based on the user's referer. Using this plugin you can show a different contact details to users coming from Google search or from social networks.

== Installation ==
1. Upload the referer-specific-contact directory to the plugins directory.
2. Activate the plugin "Referer Specific Contact" through the "Plugins" menu in WordPress.
3. Configure contact details through "Referer Specific Contact" sub-menu under "Settings" menu.
4. Use shortcodes "referer-contact-phone" for displaying phone number and "referer-contact-email" for displaying email address
